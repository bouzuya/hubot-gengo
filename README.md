# hubot-gengo

A Hubot script for responding the japanese era name.

## Usage

    $ npm install git://github.com/bouzuya/hubot-gengo.git

or

    $ # TAG is the package version you need.
    $ npm install git://github.com/bouzuya/hubot-gengo.git\#TAG
